# 安全与隐私基线（未成年人优先）

## 默认
- 学习数据默认私有；学生不可被搜索
- 建立关系：分享码 / 班级邀请码 / （可选）开启「允许被搜索」

## 授权粒度（scope）
- `progress:read`
- `metrics:read`
- `works:read`
- `code:read` (默认不含给家长，仅教师可选)

## 生命周期
- 同意后生成 grant；默认 30/90 天到期；可续期；可随时撤销
- 撤销即时生效；历史审计保留

## 审计
- 任何查看与导出都写入 audit_logs（actorId, action, targetId, ts, route, meta）

## 风控
- 搜索/申请限流；异常频次封禁
- 对外展示去标识化；家长对比只给“自家孩子 vs 班级匿名分位”
